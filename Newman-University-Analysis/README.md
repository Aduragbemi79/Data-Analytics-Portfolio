# Newman University Performance Analysis Dashboard

## Project Overview

Developed an interactive **Power BI dashboard** to analyse **Newman University’s position in the Complete University Guide (CUG) league table** and evaluate the performance metrics influencing institutional ranking. The project focused on identifying strengths, weaknesses, competitor benchmarking, and actionable recommendations to support strategic improvement.

## Business Problem

Newman University sought to better understand its league table position and identify the key performance indicators impacting ranking performance in order to improve its standing in the **2023/2024 academic year**.

## Project Objective

To analyse Newman University’s ranking position, understand the individual metrics used in the **Complete University Guide (CUG)**, and provide data-driven insights and recommendations to improve institutional performance.

## Tools & Technologies Used

* Power BI
* Python (Data Cleaning & Preparation)
* DAX
* Data Visualisation
* Excel
* Business Intelligence Reporting

## Data Preparation & Processing

* Performed **data cleaning and transformation** to improve data quality and consistency.
* Addressed missing values using statistical imputation methods for analysis accuracy.
* Conducted comparative benchmarking across universities and institutional groups.
* Developed calculated measures and KPIs to support performance evaluation.

## Key Analysis Performed

* Competitor analysis using **top 5 and bottom 5 universities** around Newman University’s ranking.
* Benchmarking against **West Midlands universities** and **Cathedral University Group institutions**.
* Comparative metric analysis on:

  * Student Satisfaction
  * Graduate Prospects
  * Degree Completion
  * Research Quality & Intensity
  * Student-Staff Ratio
  * Facilities Spend

## Key Insights

* Newman University performed strongly in **Student Satisfaction** and **Graduate Prospects (On Track)** metrics.
* Improvement opportunities were identified in:

  * Research Quality
  * Research Intensity
  * Student-Staff Ratio
  * Facilities Spending
  * Graduate Outcomes
* Competitor benchmarking highlighted areas for strategic institutional improvement.

## Recommendations

* Increase investment in **research quality and academic intensity**.
* Improve graduate employability through partnerships, internships, and career support initiatives.
* Reduce student-staff ratio through targeted academic recruitment.
* Enhance student facilities and academic services to improve overall student experience.

## Dashboard Features

* Interactive KPI visualisations
* Competitor benchmarking dashboards
* Regional comparison analysis
* Metric performance comparison
* Course and subject-level insights

## Outcome

Delivered an insight-driven Power BI solution that enabled evidence-based decision-making and provided actionable recommendations to improve Newman University’s league table performance.
